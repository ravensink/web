El equipo de RavenSinK sabe lo importante que es la privacidad de los usuarios que hacen uso del bot,
pues es algo bastante serio, así que aqui te mostramos la política de privacidad del bot, donde puedes
ver que información recolecta RavenSinK y cual no.

Nota importante : RavenSinK no solicita, comparte o vende tu información,
la información recolecta es la requerida para un buen funcionamiento del bot.
Toda la información esta almacenada indefinidamente.





RavenSinK recolecta...



1. Información proporcionada voluntariamente mediante comandos.

2. Métricas anónimas :
- Mensajes recibidos por segundo
- Comandos ejecutados por minuto
- Peticiones de la API de Discord y Eventos de Gateways.

3. Todos los elementos que hagan uso de ID'S :
- Message ID
- Channel ID
- Server ID
- User ID.




RavenSinK no recolecta...



1. Cualquier cosa no incluida en la parte de arriba.

2. Contenido de mensajes.

3. Información no util para el bot.




• El contenido de los mensajes solo es usado para el sistema antilinks y para leer el prefix, esa información se guarda durante
unos pocos segundos - Los demás son obtenidos de la API de Discord.

RavenSinK collects...



1. Information provided voluntarily through commands.2. Anonymous Metrics:

- Channel ID
- Server ID
- User ID.




RavenSinK does not collect...



1. Anything not included above.
